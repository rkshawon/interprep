#### Write a Program to use a callback function?

```javascript
function great(mgs, fn) {
  fn(mgs);
}
great("Message", (mgs) => console.log(mgs));
```

###### Output: `Message`

#### Write a Program to add a property to an object and delete a property from an object?

###### Add a property from an object:

```javascript
const obj = { name: "Kiba" };
obj.age = 21;
console.log(obj);
```

###### Output: `{ name: 'Riya', age: 21 }`

###### Delete a property from an object:

```javascript
const obj = { name: "Riya", age: 21 };
delete obj.age;
console.log(obj);
```

###### Output: `{ name: 'Riya' }`

#### Write a function which duplicates the array items?

```javascript
function dup(arr) {
  return [...arr, ...arr];
}
console.log(dup([1, 2, 3]));
```

###### Output: `[1, 2, 3, 1, 2, 3]`
