# JavaScript Arrays - Complete Guide üöÄ

#### Introduction

An **array** is a special data structure in JavaScript used to store multiple values in a **single variable**.

##### Declaring an Array

```javascript
let arr1 = [1, 2, 3]; // Using array literal ‚úÖ (Recommended)
let arr2 = new Array(1, 2, 3); // Using Array constructor ‚ùå (Less common)

// Checking if a Variable is an Array
console.log(Array.isArray(arr1)); // ‚úÖ true
console.log(typeof arr1); // ‚ùå "object" (Arrays are objects in JS)
```

#### Array Indexing & Mutability

Arrays in JavaScript are zero-indexed and mutable, meaning you can modify them after creation.

```javascript
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // "apple" (First element)
console.log(fruits[fruits.length - 1]); // "cherry" (Last element)

fruits[1] = "blueberry"; // Modify element
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

#### Array Methods

##### Adding & Removing Elements

| Method      | Operation                     |
| ----------- | ----------------------------- |
| `push()`    | Add element(s) at the end     |
| `pop()`     | Remove element from the end   |
| `unshift()` | Add element(s) at the start   |
| `shift()`   | Remove element from the start |

```javascript
let arr = [1, 2, 3];

// Push & Pop
arr.push(4); // [1, 2, 3, 4]
arr.pop(); // [1, 2, 3]

// Unshift & Shift
arr.unshift(0); // [0, 1, 2, 3]
arr.shift(); // [1, 2, 3]
```

##### Slicing & Splicing

```javascript
let nums = [10, 20, 30, 40, 50];

// Slice (Creates a new array)
let sliced = nums.slice(1, 3); // [20, 30]
console.log(nums); // [10, 20, 30, 40, 50] (Original unchanged)

// Splice (Modifies the array)
nums.splice(2, 1, 99); // Removes 1 element at index 2, adds 99
console.log(nums); // [10, 20, 99, 40, 50]
```

##### Searching in Arrays

JavaScript provides several methods for searching array elements:

```javascript
let nums = [5, 10, 15, 20, 10];

console.log(nums.indexOf(10)); // ‚úÖ 1 (First occurrence)
console.log(nums.lastIndexOf(10)); // ‚úÖ 4 (Last occurrence)
console.log(nums.includes(15)); // ‚úÖ true
console.log(nums.find((n) => n > 10)); // ‚úÖ 15 (First match)
console.log(nums.findIndex((n) => n > 10)); // ‚úÖ 2 (First matching index)
```

##### Array Iteration

######## Using forEach()

```javascript
let arr = ["a", "b", "c"];
arr.forEach((value, index) => console.log(index, value));
```

######## Using map()

```javascript
let numbers = [1, 2, 3];
let doubled = numbers.map((num) => num * 2); // [2, 4, 6]
```

##### Filtering & Reducing

```javascript
let nums = [1, 2, 3, 4, 5];

// Filter
let evens = nums.filter((n) => n % 2 === 0);
console.log(evens); // [2, 4]

// Reduce
let sum = nums.reduce((acc, curr) => acc + curr, 0);
console.log(sum); // 15
```

##### Sorting & Reversing

```javascript
let letters = ["d", "a", "c", "b"];
letters.sort(); // ["a", "b", "c", "d"]
letters.reverse(); // ["d", "c", "b", "a"]

let nums = [10, 1, 5, 3, 8];
nums.sort((a, b) => a - b); // [1, 3, 5, 8, 10] (Ascending)
```

#### Array Behaviors & Common Patterns

##### Dynamic Sizing

```javascript
let arr = [1, 2, 3];
arr[5] = 6;
console.log(arr); // [1, 2, 3, empty √ó 2, 6]
console.log(arr.length); // 6
```

##### Arrays as Objects

```javascript
let arr = [1, 2, 3];
console.log(typeof arr); // "object"
console.log(arr instanceof Array); // true
```

##### Shallow vs Deep Copy

```javascript
let original = [1, 2, 3];
let shallowCopy = original; // ‚ùå Copies reference

shallowCopy[0] = 99;
console.log(original); // [99, 2, 3]

let deepCopy = [...original]; // ‚úÖ Creates a true copy
deepCopy[0] = 100;
console.log(original); // [99, 2, 3]
console.log(deepCopy); // [100, 2, 3]
```

#### Key Takeaways

- Arrays in JavaScript are dynamic and mutable objects
- Methods like `map()`, `filter()`, `reduce()` create new arrays
- `sort()` and `reverse()` modify the original array
- Use spread operator (`[...]`) for shallow copying
- Arrays can have empty slots (sparse arrays)
