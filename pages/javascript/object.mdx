#### 1. **Introduction**

Objects in JavaScript are **collections of key-value pairs**. They allow storing multiple values, including functions.

```javascript
const person = {
  name: "Alice",
  age: 25,
  greet: function () {
    console.log(`Hello, my name is ${this.name}`);
  },
};

person.greet(); // Output: Hello, my name is Alice
```

#### 3. Accessing Object Properties

##### Dot Notation

```javascript

```

##### Bracket Notation

```javascript

```

##### Dynamic Property Access

```javascript

```

#### 4. Adding, Modifying, and Deleting Properties


#### 9. Object Freezing & Sealing

##### Freezing (Object.freeze())

Prevents adding, modifying, and deleting properties.

```javascript
const frozenObj = Object.freeze({ name: "Frozen" });
frozenObj.name = "Changed"; // ‚ùå Won't work
delete frozenObj.name; // ‚ùå Won't work
console.log(frozenObj); // { name: "Frozen" }
```

##### Sealing (Object.seal())

Prevents adding or deleting properties but allows modification.

```javascript
const sealedObj = Object.seal({ name: "Sealed" });
sealedObj.name = "Modified"; // ‚úÖ Allowed
delete sealedObj.name; // ‚ùå Won't work
console.log(sealedObj); // { name: "Modified" }
```

#### 10. Prototypes and Inheritance

##### Prototype Chain

```javascript
function Animal(name) {
  this.name = name;
}

Animal.prototype.speak = function () {
  console.log(`${this.name} makes a noise.`);
};

const dog = new Animal("Dog");
dog.speak(); // Dog makes a noise.
```

##### ES6 Class Inheritance

```javascript
class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a noise.`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks.`);
  }
}

const myDog = new Dog("Buddy");
myDog.speak(); // Buddy barks.
```

#### 12. Common Interview Questions

##### Q1: What's the difference between Object.freeze() and Object.seal()?

‚úÖ Answer:

- Object.freeze() ‚ùå No modification, No addition, No deletion.
- Object.seal() ‚úÖ Modification allowed, No addition, No deletion.

##### Q2: How do you check if an object has a property?

‚úÖ Answer:

- "key" in obj
- obj.hasOwnProperty("key")
- Object.keys(obj).includes("key")

##### Q3: What is the difference between shallow copy and deep copy?

‚úÖ Answer:

- Shallow Copy ‚Üí Copies only references (changes affect original).
- Deep Copy ‚Üí Copies everything independently.

######## Example: Shallow Copy

```javascript
const obj = { name: "Alice", details: { age: 25 } };
const shallowCopy = { ...obj };
shallowCopy.details.age = 30;
console.log(obj.details.age); // 30 (Modified)
```

###### Example: Deep Copy (Using structuredClone() in modern JS)

```javascript
const deepCopy = structuredClone(obj);
deepCopy.details.age = 40;
console.log(obj.details.age); // 30 (Unchanged)
```

#### Conclusion

- Utilize Object.freeze(), Object.seal() to control modifications.
- Understand prototypes for inheritance.
- Know shallow vs deep copy to avoid unintended modifications.
